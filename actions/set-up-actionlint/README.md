# set-up-actionlint

Install actionlint binary with a pinned version.

## Inputs

| Name      | Type   | Default  | Description                   |
| --------- | ------ | -------- | ----------------------------- |
| `version` | string | `1.7.12` | actionlint version to install |

## Usage

```yaml
- uses: cboone/gh-actions/actions/set-up-actionlint@v3.0.0
- run: actionlint
```
